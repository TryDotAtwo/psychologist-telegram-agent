Читать сайт и статьи можно без согласия. Чат, запись, связка Telegram и хранение профиля доступны только после явного согласия.

Текст согласия и политики

Согласие принято. Теперь доступны web-chat, запись и связка с Telegram.

Web-chat

Сообщения после согласия видны в dashboard как site-клиент. Telegram-отправка включается только после явной deep-link связки.

Запись

Заявка создается с источником site и попадает в dashboard.

Связать Telegram

Сайт создает одноразовый deep-link. Cookie сама по себе не связывает профиль и Telegram.

Статьи

Публикуются из dashboard после ручной проверки. В публичный JSON не попадают внутренние id и приватные поля.

Данные и прозрачность

Базовый privacy-контур: информированное согласие, отдельная public API-зона, отсутствие приватных клиентских данных в публичных ответах.

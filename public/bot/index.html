Как работает запись

Google Calendar используется как источник занятости. Если событие уже есть в Google, бот не предложит это время клиенту.

При записи бот создает событие-удержание в календаре психолога.

Кризисная инструкция

Ограничения

  • Админ API закрыт cookie-сессией или bearer token.
  • Webhook Telegram принимает только запросы с secret header.
  • Ручные заметки психолога не перетираются агентом.
  • Медицинские сведения хранятся как слова клиента, не как диагноз.

Данные не загружены.
